\subsection{Objetivos}
Los objetivos principales de este trabajo son:

\begin{enumerate}
  \item Diseñar una base de datos relacional estructurada específicamente para
        soportar modelos de IA y análisis estadístico en el dominio botánico.
  \item Integrar y normalizar datos de fuentes abiertas heterogéneas (APIs
        botánicas, datos climáticos).
  \item Desarrollar e implementar modelos estadísticos y de aprendizaje
        automático para la recomendación de plantas.
  \item Evaluar los modelos con métricas estadísticas robustas e interpretar
        los resultados de forma explicable.
  \item Demostrar que el diseño de la base de datos influye en la calidad
        predictiva de los modelos.
\end{enumerate}
